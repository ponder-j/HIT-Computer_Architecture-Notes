三项性能指标：吞吐率、加速比、效率

## 1 吞吐率

- 单位时间内流水线
- $TP_{max}$
- $TP$

计算：

$TP_{max} = \frac{1}{\Delta t_{0} }$
$TP = \frac{n}{T_{流水}}$
$T_{流水} = m\cdot \Delta t_{0} + (n-1)\Delta t_{0}$

m 段，第 i 段 $\Delta t_{i}$
$\Sigma_{i=1}^{m}\Delta t_{i} + (n-1)\cdot \max\{\Delta t_{i}\}$

### 2 加速比

$S = \frac{T_{非流水}}{T_{流水}}$

### 3 效率

$E = \frac{n \cdot t_{0}}{T_{流水}} = \frac{S}{m} = TP\Delta t_{0}$

通过时间，排空时间
